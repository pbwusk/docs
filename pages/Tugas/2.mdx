# Tugas 02 - HTML

Setelah mempelajari HTML, sekarang saatnya untuk mempraktekkan apa yang telah dipelajari.

Tugas kali ini adalah membuat sebuah halaman web sederhana

1. Tugas pertama, kalian harus membuat sebuah halaman berisikan biodata diri kalian sendiri. Biodata yang harus ditampilkan adalah:

```
Nama lengkap:
Foto:
NIM:
Jurusan:
Fakultas:
Universitas:
```

2. Di bawah biodata, kalian harus membuat sebuah tabel yang berisikan mata kuliah yang kalian ambil pada semester ini! Tabel tersebut harus berisikan:

```
Nama Mata Kuliah:
SKS:
Dosen Pengampu:
Ruangan:
Waktu & Hari:
```

## Petunjuk

<Callout type="warning" emoji="⚠️">
  Tugas ini tidak perlu menggunakan styling, hanya menggunakan HTML saja.
  Styling hanya boleh digunakan untuk mengatur ukuran gambar saja.
</Callout>
Dalam mengerjakan tugas ini, kalian menggunakan repository yang sudah kalian buat
pada tugas sebelumnya. Lalu, silahkan buat folder baru pada repository tersebut dengan
nama folder `tugas-02`, kemudian buatlah file `index.html` pada folder tersebut dan
silahkan lakukan coding pada file tersebut.
