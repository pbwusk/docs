# Tugas 03 - CSS

Setelah mempelajari CSS, sekarang saatnya untuk mempraktekkan apa yang telah kita pelajari


Tugas kali ini adalah menstyling dan menambahkan sesuatu pada web yang sudah dibuat pada tugas 02 - HTML

1. Langkah pertama, kalian harus membuat Header dan Footer. Pastikan tulisan Header dan Footer memiliki warna yang berbeda dari background body serta tulisan untuk header dan footer berada di tengah.

<Callout type="warning" emoji="⚠️">
    hint: Gunakan tag ``` <header></header> ``` untuk membuat Header dan gunakan tag ``` <footer></footer> ``` untuk membuat footer


    - Isi content pada header : "DAFTAR MATA KULIAH SAYA"
    - Isi content pada footer : "© Made By ```<nama_anda> ``` "
</Callout>

2. Langkah Kedua, Silahkan lakukan styling untuk tabel yang sudah kalian buat pastikan background warna pada header berbeda dengan datanya.


3. Langkah Ketiga, Silahkan buat card yang didalam card tersebut terdapat biodata kalian beserta foto kalian.

<Callout type="warning" emoji="⚠️">
Pastikan posisi card berada selalu disebelah kanan!
agar posisi card selalu berada di sebelah kanan kalian bisa menambahkan property "float" dan untuk value dalam property float tersebut adalah "right"

Tidak harus menggunakan cara ini kalian bisa menggunakan cara lain. Untuk mengetahui cara yang lain silahkan mencoba-coba atau bereksplorasi sendiri
</Callout>

4. Pastikan dalam kalian melakukan styling pada CSS menggunakan eksternal CSS


Kurang lebih tampilan web setelah kalian melakukan styling harusnya menjadi seperti [ini](https://pbwusk.github.io/biodata/)


<Callout type="warning" emoji="⚠️">
Note: Untuk warnya tidak perlu sama persis bisa memakai warna yang berbeda selain warna putih
