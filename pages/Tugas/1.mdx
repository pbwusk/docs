# Tugas 01 - Git

Setelah mempelajari materi tentang Git, sekarang saatnya untuk mempraktekkan apa yang telah dipelajari. Tugas kali ini adalah membuat repository baru di akun github masing-masing dan melakukan commit.

## Persiapan

Sebelum memulai tugas, pastikan bahwa mahasiswa telah menginstall Git dan telah membuat akun github. Jika belum, silahkan ikuti panduan berikut:

- [Install Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
- [Membuat Akun Github](https://github.com/)

## Membuat Repository

Mahasiswa diahruskan membuat repository baru dengan nama `lab-pbw-b-{nama}` di akun github masing-masing. Repository ini akan digunakan untuk menyelesaikan tugas-tugas selama semester ini.

## Menambahkan File

Setelah membuat repository, mahasiswa diminta untuk membuat file dan melakukan commit terhadap repository tersebut. Mahasiswa membuat file `README.md` yang berisi nama dan NIM mahasiswa. Contoh isi file `README.md`:

```md copy
# Ini merupakan repository tugas praktikum Pemrograman Berbasis Web INF 224 USK 2023

## Data Diri

Nama:
NPM:
```

## Commit

<Callout type="info" emoji="⚠️">
  Mahasiswa diharapkan mempelajari Conventional Commits, dan melakukan commit
  dengan format yang sesuai. Mahasiswa dapat mempelajarinya dengan membaca
  dokumentasinya pada link di bawah.
  <a

  >
    Klik untuk membaca

## Push

Jika sudah add file dan melakukan commit, mahasiswa melakukan push ke repository github masing-masing. Mahasiswa dapat melakukan push dengan cara berikut:

```bash
git push origin main
```

## Penilaian

- 20 poin: Repository sudah dibuat
- 20 poin: File `README.md` sudah dibuat dan mengikuti isi yang diperintahkan
- 15 poin: Commit sudah dilakukan
- 15 poin: Push sudah dilakukan
- 30 poin: Commit sudah menggunakan format Conventional Commits
