# List di HTML

List adalah kumpulan item yang disusun secara berurutan. List dapat dibuat dengan menggunakan tag `<ul>` dan `<ol>`. List juga dapat dibuat dengan menggunakan tag `<li>`.

## Unordered List

List tidak berurutan (Unordered List) menggunakan tag `<ul>` dan `<li>`. Contoh:

```html copy filename="ul-list.html"
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

![Unordered List](/images/web/unordered.jpeg)

## Ordered List

List berurutan (Ordered List) menggunakan tag `<ol>` dan `<li>`. Contoh:

```html copy filename="ol-list.html"
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>
```
