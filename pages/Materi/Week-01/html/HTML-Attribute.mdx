# Atribut HTML

Atribut HTML adalah atribut yang digunakan untuk memberikan informasi tambahan pada elemen HTML. Atribut HTML dapat berupa atribut khusus, atribut global, dan atribut event.

## Atribut Khusus

Atribut khusus adalah atribut yang hanya dapat digunakan pada elemen tertentu. Contoh atribut khusus adalah atribut `src` pada elemen img, yang digunakan untuk menentukan sumber gambar yang akan ditampilkan.

Contoh lain atribut khusus adalah atribut `href` pada elemen a, yang digunakan untuk menentukan sumber halaman yang akan dituju ketika elemen a diklik.

## Atribut Global

Atribut global adalah atribut yang dapat digunakan pada semua elemen HTML. Contoh atribut global adalah atribut `class` dan `id`, yang digunakan untuk menentukan nama kelas dan nama id dari elemen.

## Inline CSS

<Callout type="info" emoji="⚠️">
  CSS akan dipelajari lebih dalam pada materi pekan selanjutnya.
</Callout>

Inline CSS adalah cara untuk menambahkan style pada elemen HTML. Inline CSS dapat ditambahkan pada elemen HTML dengan menambahkan atribut `style` pada elemen tersebut.

Contoh:

```html filename="inlinecss.html" copy
<p style="color: red;">Ini adalah paragraf</p>
```
