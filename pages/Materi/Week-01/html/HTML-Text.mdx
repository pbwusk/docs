# Teks pada HTML

Untuk menampilkan teks pada halaman yang kita kerjakan, kita menggunakan _tag_ teks yang sudah disampaikan pada materi sebelumnya. Pada halaman ini, akan dibahas mengenai _tag_ untuk teks tersebut.

## Heading

Heading, adalah sebuah _tag_ yang digunakan untuk membuat judul pada halaman web. Heading memiliki 6 tingkatan, yaitu:

- Heading 1
- Heading 2
- Heading 3
- Heading 4
- Heading 5
- Heading 6

Semakin ke bawah maka semakin kecil ukuran teksnya. Contoh penggunaan _tag_ Heading pada HTML:

```html copy filename=heading.html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

## Paragraph

Jika Header digunakan untuk sebuah judul, maka paragraph, adalah sebuah _tag_ yang digunakan untuk membuat paragraf pada halaman web. Contoh penggunaan _tag_ Paragraph pada HTML:

```html copy filename="paragraph.html"
<p>Paragraf di HTML</p>
```

![Paragraph Examples](/images/web/paragraph.jpeg)

## Hyperlink

Hyperlink atau tag `<a>`, adalah sebuah _tag_ yang digunakan untuk membuat link pada halaman web. Contoh penggunaan _tag_ Hyperlink pada HTML:

tag `<a>` memiliki atribut `href` yang berfungsi untuk menentukan link yang akan dibuka ketika link tersebut diklik. Contoh penggunaan atribut `href` pada tag `<a>`:

```html copy filename="hyperlink.html"
<a href="https://gdsc.community.dev/universitas-syiah-kuala/">GDSC USK</a>
```
