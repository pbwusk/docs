# Body HTML

Body merupakan bagian penting pada HTMl, karena body merupakan bagian yang akan ditampilkan pada browser. Body dapat diisi oleh berbagam _tag_ HTML, seperti _tag_ `<h1>`, `<p>`, `<img>`, `<table>`, dan lain-lain. Berikut ini adalah contoh penggunaan _tag_ pada body.

## Contoh Penggunaan Tag pada Body

```html copy filename="body.html"
<!DOCTYPE html>
<html>
  <body>
    <h1>Contoh Penggunaan Tag pada Body</h1>
    <p>Ini adalah contoh penggunaan tag pada body.</p>
    <table>
      <tr>
        <th>Firstname</th>
        <th>Lastname</th>
        <th>Age</th>
      </tr>
      <tr>
        <td>Jill</td>
        <td>Smith</td>
        <td>50</td>
      </tr>
      <tr>
        <td>Eve</td>
        <td>Jackson</td>
        <td>94</td>
      </tr>
    </table>
  </body>
</html>
```
