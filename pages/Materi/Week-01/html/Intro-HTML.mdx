# Pengantar HTML

## Pengertian & Gambaran Dasar HTML

HyperText Markup Language atau yang disingkat menjadi HTML adalah sebuah bahasa markup yang digunakan untuk mengembangkan website.
File HTML ditandai dengan ekstensi `.html` pada akhir nama file. Contoh nama file HTML adalah `index.html`. Untuk Menjalankan file HTML dibutuhkan sebuah `Browser` atau `Peramban`. Browser akan menampilkan isi dari file HTML yang sudah dibuat atau dijalankan.

## Cara Membuat File HTML

Untuk membuat file HTML, kita bisa menggunakan text editor seperti Notepad, Notepad++, Sublime Text, Visual Studio Code, Atom, dll. Setelah itu, kita bisa menyimpan file HTML dengan ekstensi `.html` pada akhir nama file. Contoh nama file HTML adalah `index.html`.

## Struktur Dasar HTML

Berikut adalah contoh struktur dasar dari HTML.

```html copy filename="index.html"
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>
```

Untuk selengkapnya, akan dibahas pada materi selanjutnya yaitu [Tag Pada HTML](./HTML-Tag).
