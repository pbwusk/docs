# Head HTML

Head pada HTML merupakan bagian yang berada di dalam tag <head> dan </head>. Head HTML berisi informasi mengenai halaman web yang sedang dibuka. Contoh informasi yang dapat ditulis di dalam head HTML adalah judul halaman, deskripsi halaman, dan keyword halaman. Selain itu, head HTML juga dapat berisi script, style, dan link.

## Title

Title atau judul halaman merupakan informasi yang paling penting yang harus ditulis di dalam head HTML. Judul halaman akan ditampilkan di bagian atas jendela browser. Judul halaman juga akan ditampilkan di daftar bookmark dan di hasil pencarian mesin pencari.

Contoh judul halaman:

```html {2} copy filename="title.html"
<head>
  <title>Belajar HTML</title>
</head>
```

Style adalah kode CSS yang ditulis di dalam halaman web. Style dapat ditulis di dalam tag `<style>` dan `</style>` atau di dalam tag `<link rel="stylesheet" href="namafile.css">`.

Contoh style:

```html {2} copy filename="style2.html"
<head>
  <link rel="stylesheet" href="style.css" />
</head>
```

## Meta

Meta adalah informasi tambahan yang ditulis di dalam head HTML. Meta dapat berupa deskripsi halaman, keyword halaman, dan informasi mengenai author halaman.

Contoh meta:

```html {2} copy filename="meta.html"
<head>
  <meta name="description" content="Belajar HTML" />
  <meta name="keywords" content="HTML, Belajar HTML" />
  <meta name="author" content="GDSC USK" />
</head>
```

## Link

Link adalah informasi mengenai sumber data lain yang digunakan di dalam halaman web. Link dapat berupa gambar, video, dan dokumen lainnya.

Contoh link:

```html {2} copy filename="link.html"
<head>
  <link rel="icon" href="logo.png" />
</head>
```
