## Penggunaan Tag

Agar _tag_ dapat berjalan dengan semestinya, tag tersebut harus dibuka kemudian ditutup. Bagaimana maksudnya? Berikut merupakan contoh penggunaan tag:

```html copy
<h1>Ini Tag untuk Header-1!</h1>
```

Pada kode di atas, dapat dilihat terdapat dua tag yaitu `<h1>` dan `</h1>`. Tag `<h1>` merupakan tag pembuka, sedangkan tag `</h1>` merupakan tag penutup. Tag penutup ditandai dengan adanya tanda `/` sebelum tag pembuka.

## Self-Closing Tag

Tidak semua tag harus dibuka dan ditutup. Ada beberapa tag yang hanya perlu dibuka saja, yaitu tag yang disebut dengan _self-closing tag_. Berikut merupakan contoh penggunaan _self-closing tag_:

```html copy
<img src="gambar.jpg" alt="Gambar" />
```

Pada kode di atas, tag `<img>` merupakan _self-closing tag_. Tag tersebut hanya perlu dibuka saja, tidak perlu ditutup.
