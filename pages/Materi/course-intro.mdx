Selamat datang di laman pembelajaran Pemrograman Web. Pada laman ini, anda akan mempelajari tentang dasar-dasar pemrograman web. Pemrograman web adalah suatu proses untuk membuat sebuah website.

## Apa saja yang perlu dipersiapkan?

Peserta diharapkan dapat mempersiapkan hal berikut

1. Code Editor, peserta dapat mengunduh melalu link berikut [Visual Studio Code](https://code.visualstudio.com/) atau dapat menggunakan layanan online melalui [Replit](https://replit.com)
2. Browser, peserta dapat menggunakan browser apapun, namun disarankan menggunakan [Google Chrome](https://www.google.com/chrome/)

## Apa saja yang akan dipelajari?

Peserta akan mempelajari seputar HTML, CSS, JavaScript.
