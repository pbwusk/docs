# Text Editor

## Apa itu Text Editor?

Text editor adalah sebuah aplikasi yang digunakan untuk mengedit sebuah file teks. Biasanya text editor digunakan untuk mengedit file konfigurasi, file HTML, file PHP, file CSS, file Javascript, dan lain-lain. Text editor juga digunakan untuk mengedit file teks biasa seperti file txt, file log, dan lain-lain.

Text editor juga digunakan untuk mengedit file teks yang berisi kode program seperti file PHP, file Javascript, file CSS, dan bahasa pemrograman lain.

## Jenis-jenis Text Editor

Ada banyak sekali text editor yang tersedia di internet. Namun, untuk kebutuhan pemula, saya sarankan menggunakan text editor yang sederhana dan mudah digunakan. Berikut adalah beberapa text editor yang bisa digunakan untuk kebutuhan pemula.

- Notepad (Windows)
- TextEdit (Mac OS)
- Gedit (Linux)
- Sublime Text
- Atom
- Visual Studio Code
- Brackets
- Notepad++

## Menggunakan Text Editor

Pada tutorial ini, saya menggunakan text editor Visual Studio Code untuk mengedit file teks. Untuk menggunakan text editor Visual Studio Code, silahkan download terlebih dahulu di [sini](code.visualstudio.com/download).

Setelah selesai download, silahkan install Visual Studio Code. Setelah selesai install, silahkan buka Visual Studio Code.

Pada Visual Studio Code, terdapat beberapa menu yang bisa digunakan untuk mengedit file teks. Berikut adalah beberapa menu yang ada pada Visual Studio Code.

- File
- Edit
- Selection
- View
- Go
- Run
- Terminal
- Help
