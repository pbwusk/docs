# Hyper Text Markup Language (HTML)

## Apa itu HTML?

HTML pertama kali diperkenalkan oleh Tim Berners-Lee pada tahun 1991. HTML merupakan bahasa Markup yang menjadi standar dalam pembuatan Website. Html tidak dianggap sebagai bahasa pemrograman karena tidak bisa melakukan proses komputasi.

Html hanya bisa melakukan proses markup, yaitu memberikan tanda pada teks untuk menentukan bagaimana teks tersebut akan ditampilkan di layar. Html juga tidak bisa melakukan proses penyimpanan data, karena data yang disimpan hanya bisa ditampilkan di layar.

## Cara Kerja HTML

Cara kerja HTML adalah dengan menuliskan kode-kode html pada sebuah file teks. File teks tersebut kemudian disimpan dengan ekstensi .html. Setelah itu file tersebut diakses melalui browser. Browser akan membaca file html tersebut dan menampilkan hasilnya di layar.

## Element HTML

### Tag 

Ada banyak sekali tag html yang bisa digunakan. Tag HTML terdiri dari tag pembukan dan tag penutup

```
<tag_pembuka>isi tag</tag_penutup>
```

Contoh tag 

```
<h1>Ini adalah tag h1</h1>
```

### Atribut 

Atribut adalah informasi tambahan yang diberikan pada tag html. Atribut html dituliskan di dalam tag pembuka.

```
<tag_pembuka atribut="nilai atribut">isi tag</tag_penutup>
```

Contoh atribut 

```
<h1 class="judul">Ini adalah tag h1</h1>
```

### Comment

Comment adalah bagian dari kode html yang tidak akan ditampilkan di layar. Comment digunakan untuk memberikan keterangan pada kode html.

```
<!-- ini adalah comment -->
```

## Struktur Dasar HTML

Dokumen html memiliki struktur dasar seperti berikut

```
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <-- isi dokumen html -->
</body>
</html>
```
